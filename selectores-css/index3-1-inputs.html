Selectores combinatorios

Ejemplo 1: Todos los elementos div que están dentro del main

                    main div {
                        background-color: aqua; /* Changed to aqua for better readability */
                    }
Elemento div

Ejemplo 2: Cualquier div que esté dentro de la etiqueta main justo después

                    main > div {
                        background-color: rgb(42, 54, 42);
                    }
Elemento div

Ejemplo 3: Seleccionar el primer elemento en secuencia del mismo nivel (el primer div después de main)

                    main + div {
                        background-color: yellow;
                    }
Elemento div

Ejemplo 4: Seleccionar todos los elementos en secuencia del mismo nivel (los h3 después de h2)

                    h2 ~ h3 {
                        background-color: aqua;
                    }

Selectores avanzados

Ejemplo 5: Seleccionar elementos cuyo atributo 'value' contiene la palabra "maria"

                    [value*="maria"] {
                        font-size: 24px;
                    }

Ejemplo 6: Seleccionar elementos cuyo atributo 'value' comienza exactamente con el valor "maria"

                    [value|="maria"] {
                        background-color: violet;
                    }

Ejemplo 7: Seleccionar elementos cuyo atributo 'value' comienza con el valor "maria"

                    [value^="maria"] {
                        background-color: violet;
                    }

Ejemplo 8: Seleccionar elementos cuyo atributo 'value' termina con "maria"

                    [value$="maria"] {
                        background-color: violet;
                    }

Ejemplo 9: Seleccionar elementos cuyo atributo 'value' incluye el valor "ia"

                    [value*="ia"] {
                        background-color: violet;
                    }
